# TypeScript

## What is TypeScript?

TypeScript is a free, open-source programming language built on top of JavaScript. It adds static type checking and other advanced features to make development more robust and scalable.

## Key Features

- **Static Typing**: Catch errors at compile-time rather than runtime
- **Object-Oriented Programming**: Classes, interfaces, and inheritance
- **Advanced Features**: Generics, decorators, and enums
- **Backward Compatible**: All valid JavaScript is valid TypeScript
- **Compilation**: Compiles to clean, readable JavaScript

## Installation

```bash
npm install -g typescript
```

## Basic Example

```typescript
function greet(name: string): string {
    return `Hello, ${name}!`;
}

greet("World");
```

## When to Use

- Large-scale applications
- Team projects requiring code clarity
- Projects needing strong error prevention
- Complex business logic
